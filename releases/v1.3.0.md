# v1.3.0 — Enterprise Templates, DVC, Observability Parity

**Release Date**: April 2026

## Highlights

This release fills every enterprise gap identified during a comprehensive template audit, ports battle-tested assets from the [ML-MLOps Portfolio](https://github.com/DuqueOM/ML-MLOps-Portfolio), and ensures all documentation reflects the current state.

## Upgrade Notes

No breaking changes. All new files are additive. Existing scaffolded services are unaffected.

```bash
git pull origin main
# New services will automatically get DVC, integration tests, and extra dirs
./templates/scripts/new-service.sh MyService my_service
```
