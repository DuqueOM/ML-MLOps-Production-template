# v1.0.0 — Initial Release

**Released: 2026-04-15**

The first public release of the ML-MLOps Production Template.

## Highlights

- **Agentic system** with 10 context-aware rules, 8 operational skills, 8 slash-command workflows
- **Multi-IDE support**: Windsurf Cascade, Claude Code, Cursor — all sharing invariants from `AGENTS.md`
- **Complete service template**: FastAPI + async inference + SHAP + Prometheus + Pandera + quality gates
- **Multi-cloud K8s**: GKE + EKS with Kustomize overlays, Workload Identity / IRSA
- **Terraform IaC**: GCP (GKE, GCS, Artifact Registry) + AWS (EKS, OIDC, IAM)
- **CI/CD templates**: 6 GitHub Actions workflows (CI, infra CI, deploy GCP/AWS, drift, retrain)
- **Monitoring**: Prometheus alerts (P1-P4) + Grafana dashboard + drift CronJob
- **12 anti-pattern detectors** (D-01 to D-12) encoded in all IDE configurations
- **Documentation templates**: ADR, runbook, model card, service README, dependency analysis
- **Open-source maturity**: SECURITY.md, CONTRIBUTING.md, CODE_OF_CONDUCT.md, issue templates, PR template, dependabot
