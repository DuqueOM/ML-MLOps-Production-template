# Examples Directory Rules

These are educational, simplified demonstrations — NOT the full production template.

- Keep examples self-contained (pip install only, no Docker/K8s/Terraform required)
- Use synthetic or public datasets only
- Keep each example runnable in < 5 minutes
- Show key patterns (async inference, SHAP, quality gates) with minimal boilerplate
- The full production templates are in `templates/`, not here
